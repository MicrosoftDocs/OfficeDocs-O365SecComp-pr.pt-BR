## <a name="how-duplicate-messages-are-identified"></a><span data-ttu-id="65a5f-110">Como as mensagens duplicadas são identificadas</span><span class="sxs-lookup"><span data-stu-id="65a5f-110">How duplicate messages are identified</span></span>

<span data-ttu-id="65a5f-111">As ferramentas de descoberta eletrônica do Office 365 usam uma combinação das seguintes propriedades de email para determinar se uma mensagem é uma duplicata:</span><span class="sxs-lookup"><span data-stu-id="65a5f-111">Office 365 eDiscovery tools use a combination of the following email properties to determine whether a message is a duplicate:</span></span>
  
- <span data-ttu-id="65a5f-112">**InternetMessageId** -esta propriedade especifica o identificador de mensagem da Internet de uma mensagem de email, que é um identificador global exclusivo que se refere a uma versão específica de uma mensagem específica.</span><span class="sxs-lookup"><span data-stu-id="65a5f-112">**InternetMessageId** - This property specifies the Internet message identifier of an email message, which is a globally unique identifier that refers to a specific version of a specific message.</span></span> <span data-ttu-id="65a5f-113">Essa ID é gerada pelo programa cliente de email do remetente ou pelo sistema de email host que envia a mensagem.</span><span class="sxs-lookup"><span data-stu-id="65a5f-113">This ID is generated by the sender's email client program or host email system that sends the message.</span></span> <span data-ttu-id="65a5f-114">Se uma pessoa enviar uma mensagem para mais de um destinatário, a ID de mensagem da Internet será a mesma para cada instância da mensagem.</span><span class="sxs-lookup"><span data-stu-id="65a5f-114">If a person sends a message to more than one recipient, the Internet message ID will be the same for each instance of the message.</span></span> <span data-ttu-id="65a5f-115">Revisões subsequentes à mensagem original receberão um identificador de mensagem diferente.</span><span class="sxs-lookup"><span data-stu-id="65a5f-115">Subsequent revisions to the original message will receive a different message identifier.</span></span> 
    
- <span data-ttu-id="65a5f-116">**ConversationTopic** -propriedade especifica o assunto do thread de conversa de uma mensagem.</span><span class="sxs-lookup"><span data-stu-id="65a5f-116">**ConversationTopic** - his property specifies the subject of the conversation thread of a message.</span></span> <span data-ttu-id="65a5f-117">O valor da propriedade **ConversationTopic** é a cadeia de caracteres que descreve o tópico geral da conversa.</span><span class="sxs-lookup"><span data-stu-id="65a5f-117">The value of the **ConversationTopic** property is the string that describes the overall topic of the conversation.</span></span> <span data-ttu-id="65a5f-118">Uma conservação consiste em uma mensagem inicial e todas as mensagens enviadas em resposta à mensagem inicial.</span><span class="sxs-lookup"><span data-stu-id="65a5f-118">A conservation consists of an initial message and all messages sent in reply to the initial message.</span></span> <span data-ttu-id="65a5f-119">As mensagens dentro da mesma conversa têm o mesmo valor para a propriedade **ConversationTopic** .</span><span class="sxs-lookup"><span data-stu-id="65a5f-119">Messages within the same conversation have the same value for the **ConversationTopic** property.</span></span> <span data-ttu-id="65a5f-120">O valor dessa propriedade normalmente é a linha de assunto da mensagem inicial que gerou a conversa.</span><span class="sxs-lookup"><span data-stu-id="65a5f-120">The value of this property is typically the Subject line from the initial message that spawned the conversation.</span></span> 
    
- <span data-ttu-id="65a5f-121">**BodyTagInfo** -esta é uma propriedade interna do repositório do Exchange.</span><span class="sxs-lookup"><span data-stu-id="65a5f-121">**BodyTagInfo** - This is an internal Exchange store property.</span></span> <span data-ttu-id="65a5f-122">O valor dessa propriedade é calculado com a verificação de vários atributos no corpo da mensagem.</span><span class="sxs-lookup"><span data-stu-id="65a5f-122">The value of this property is calculated by checking various attributes in the body of the message.</span></span> <span data-ttu-id="65a5f-123">Essa propriedade é usada para identificar as diferenças no corpo das mensagens.</span><span class="sxs-lookup"><span data-stu-id="65a5f-123">This property is used to identify differences in the body of messages.</span></span> 
    
<span data-ttu-id="65a5f-124">Durante o processo de exportação de descoberta eletrônica, essas três propriedades são comparadas a cada mensagem que corresponde aos critérios de pesquisa.</span><span <span data-ttu-id="65a5f-130">Os gráficos a seguir mostram como as mensagens duplicadas são exibidas nos relatórios **Results. csv** e **manifest. xml** exportados com os resultados da pesquisa.</span><span class="sxs-lookup"><span data-stu-id="65a5f-130">The following graphics show how duplicate messages are displayed in the **Results.csv** and **Manifest.xml** reports that are exported with the search results.</span></span> <span data-ttu-id="65a5f-131">Esses relatórios não incluem as propriedades de email descritas anteriormente, que são usadas no algoritmo de eliminação de duplicação.</span><span class="sxs-lookup"><span data-stu-id="65a5f-131">These reports don't include the email properties previously described, which are used in the de-duplication algorithm.</span></span> <span data-ttu-id="65a5f-132">Em vez disso, os relatórios incluem a propriedade de **identidade de item** atribuída a itens pelo repositório do Exchange.</span><span class="sxs-lookup"><span data-stu-id="65a5f-132">Instead, the reports include the **Item Identity** property that is assigned to items by the Exchange store.</span></span>

## <a name="limitations-of-the-de-duplication-algorithm"></a><span data-ttu-id="65a5f-139">Limitações do algoritmo de eliminação de duplicação</span><span class="sxs-lookup"><span data-stu-id="65a5f-139">Limitations of the de-duplication algorithm</span></span>
